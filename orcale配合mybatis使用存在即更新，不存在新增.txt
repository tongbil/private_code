
merge into t_sys_city t1
        USING (select '1' as a
               from dual) t2
        on (t1.cityid = t2.a)
        when matched then
            update
            set t1.cname = '汤彪'
        when not matched then
            insert (cityid,cname,provinceid,cityno)
            VALUES ('1', '1','1', '1')

 t1 指的是要插入的表的别名。 在Using中查出你想要唯一标识的主键或联合主键。
